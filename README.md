<h1>사용 라이브러리</h1>
<ESP8266WiFi.h>
<ESP8266WebServer.h>

<h1>사용 함수</h1>
<h3>scanAndConnectNetwork()</h3>
  주변 WiFi를 검색하여 HTML 코드로 저장하는 작업을 수행.

<h3>printList()</h3>
  주변 WiFi를 검색하여 리스트를 출력하는 함수. WiFi의 이름, 신호강도, 암호화 타입을 읽기 좋게 출력함.

<h3>resultNetwork()</h3>
  선택한 WiFi를 연결했을 때 결과가 나오는 함수.
